Lazarus Manor

https://github.com/RolandMarchand/lazarus-manor

I wrote a blog for an online character named Lazarus Overlook where he discusses art and programming. I designed the website with Hugo, the static website generator.

Moolisp

https://github.com/RolandMarchand/moolisp

Moolisp is a simple Lisp-like language implemented in C. It is a project created as part of the "Make a Lisp" challenge, which challenges participants to create their own Lisp interpreter in any language.

Chroma Canvas

https://github.com/RolandMarchand/chroma-canvas

This project is a simplified version of the popular r/Place platform, allowing users to collaboratively create a shared canvas by placing pixels on a grid. It is a fun and interactive way for users to express their creativity and build a collaborative digital artwork.
